### 2. NLP Microservice API for Text Analysis

- **Description**:  
  Built a FastAPI-based NLP microservice that supports multiple text processing modes, including sentiment analysis, named entity recognition (NER), POS tagging, dependency parsing, lemmatization, and more.

- **Key Features**:  
  - Accepts both single and dual text inputs (supports individual and comparative analysis)  
  - Supports multiple NLP modes via dynamic route handling  
  - Returns clean JSON responses for easy frontend integration  
  - Provides endpoints to check available processing modes and active pipelines  
  - Includes CI/CD setup with GitHub Actions and Docker for automated testing and deployment  

- **Libraries & Tools**:  
  `spaCy`, `TextBlob`, `spaCyTextBlob`, `FastAPI`, `Uvicorn`, `Docker`, `GitHub Actions`

- **Use Case**:  
  Designed for lightweight, real-time NLP processing with extensible architecture to support future modes or models.

- **Repository**:  
  [Repository](https://github.com/JPP-J/nlp1-spacy)
